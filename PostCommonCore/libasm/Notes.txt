Les Registres en 64 bits:
RAX, RBX, RCX, RDX
RSI, RDI, RBP, RSP
R8, R9, R10, R11
R12, R13, R14, R15

Conventions d’appel (calling conventions)
En 32 bits :

Les paramètres sont passés sur la pile

En 64 bits :
Les premiers paramètres sont passés dans les registres
Linux : RDI, RSI, RDX, RCX, R8, R9
Windows : RCX, RDX, R8, R9

➡️ Moins d’accès mémoire → appels de fonctions plus rapides.

Valeur de retour	            RAX
Used to store base pointer      RBP
Used to store the stack pointer RSP
After 6 args --> goes on the stack. In 32 bits all arguments are on the stack. Return value is EAX.


The stack
