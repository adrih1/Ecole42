Elf64_Ehdr

e_ident : Le tableau de 16 octets contenant les Magic Bytes (que tu as déjà vérifiés) et la classe (32 ou 64 bits).
e_entry : L'adresse virtuelle du point d'entrée actuel (ex: 0x401040). C'est cette valeur que tu devras sauvegarder puis modifier.
e_phoff (Program Header Offset) : L'endroit exact dans le fichier où commence la table des segments.
e_phnum : Le nombre d'entrées dans cette table (ton nombre d'itérations pour la boucle).
e_phentsize : La taille d'une entrée dans la table (64 octets en général pour de l'ELF64).
e_shoff / e_shnum : Idem, mais pour la table des Sections (utile si tu veux localiser précisément .text).
